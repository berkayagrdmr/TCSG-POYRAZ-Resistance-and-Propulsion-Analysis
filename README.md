
# TCSG-POYRAZ Resistance and Propulsion Analysis

This repository contains the numerical analysis and visualization scripts for the propulsion system design of the **51-meter Fast Patrol Boat, TCSG-POYRAZ**.

## Key Features
* **Resistance Estimation:** Calculated using Holtrop-Mennen and Maxsurf Resistance methods.
* **Propulsion Selection:** Powered by twin **MTU 20V 4000 M73L** engines (3600 kW each).
* **Propeller Optimization:** Wageningen B-Series propeller (Z=4) implementation with Python.
* **Performance:** Optimized for a target service speed of **27 knots**.

## Tools Used
* **Python:** NumPy for calculations, Matplotlib for high-quality technical plotting.
* **Maxsurf:** For 3D hull modeling and resistance simulations.
